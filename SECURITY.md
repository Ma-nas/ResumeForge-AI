# Security Policy

## Security Model

ResumeForge AI is a **fully client-side web application**. 
- It does **not** run on a remote server.
- It does **not** collect, store, or transmit your resume details, profile information, or job descriptions.
- Your Google Gemini API key is stored locally in your browser's `localStorage` and is only sent directly to Google's official Generative Language API endpoint (`https://generativelanguage.googleapis.com`).

## Reporting a Vulnerability

If you discover a security vulnerability or have concerns regarding how your API keys or data are handled:
1. Please do **not** open a public GitHub issue.
2. Email the maintainer directly at `manasmishra131005@gmail.com` with the details.
3. We will respond within 48 hours to assess the report and coordinate a fix.
